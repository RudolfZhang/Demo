After resetting the account, join the computer to the domain.

To join a computer to an AD DS domain, three conditions must be met:

  • You must have appropriate permissions on the computer object that allows you to join a physical computer with the same name to the domain.
  • You must be a member of the local Administrators group on the computer. This allows you to change the computer’s domain or workgroup membership.
  • You must have delegated rights to add computers to the domain or you must not have exceeded the maximum number of computer accounts that users, by default, can add to the domain. By default, users can add a maximum of 10 computers to the domain [1].

[1] The machine account quota determines the maximum number of computer accounts you can add to the domain. You can modify the value of this attribute, MS-DS-MachineAccountQuota, by using the Active Directory Service Interfaces Editor (ADSI Edit) snap-in.
