Using GPO filtering to help find the setting you need.

Most of the nodes contain multiple subfolders that enable you to organize settings even further into logical groupings. Even with this organization, finding the setting that you need might be a daunting task.

To help you locate settings you can filter the entire list of settings in either the Computer or the User section. Filter Options include: Managed, Configured, Commented, and Keyword. You can also combine multiple criteria. For example, you could filter to find all the configured settings that apply to Internet ExplorerÂ 10 by using the keyword ActiveX.

There is no a practical exercise for group policy filtering, but it is a great way to find things. In the lab, take a minute to check it out.
